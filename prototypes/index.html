// NOTES

5 个 HTML 是完全独立的，不依赖 npm 启动。直接双击 prototypes/index.html，或在浏览器地址栏输入 file:///e:/Users/Shiping/daily-assistant/prototypes/direction-1-linear.html。

原型用的是真实组件位置（侧边栏、Hero、计划、统计、记录、时段、显示器），但视觉是完全独立的，方便你专注感受"设计语言"而不是被现有功能束缚。

选定方向 + 配色后，我会基于此在真实 Vue 项目里改 token + 关键组件，预计分 2 轮迭代：第一轮换 token + 字体 + 主页 4 个 section，第二轮改造所有 view 的细节和交互。
